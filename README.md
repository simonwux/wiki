# meteor-wiki
a simple meteor wiki search application
# Requires

Node
meteor
react

# Usage

Clone the repo, then open the terminal on the folder created and run

```
git clone https://github.com/simonwux/wiki.git
cd wiki
meteor npm install
meteor reset
meteor
```
